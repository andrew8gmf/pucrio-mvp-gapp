# MVP: TFT Analytics Explorer

Este repositório contém a entrega final da Sprint de Gestão Ágil de Projetos e Produtos. O projeto consiste na ideação, concepção e planejamento ágil de uma plataforma de análise de dados para jogadores de Teamfight Tactics (TFT).

## Visão do Produto
O TFT Analytics Explorer é um dashboard de dados focado em entregar histórico rápido de partidas e análise de composições em tempo real para jogadores competitivos, consumindo dados oficiais da API da Riot Games. 

Diferente de portais de jogos genéricos e sobrecarregados, nossa proposta foca em uma jornada direta de análise pós-partida com baixíssima latência e compliance técnico estrito.
